### Projeto - Top 5 Musicas - Tião Carreiro e Pardinho
## Requisitos
- React e Laravel 11 + Banco de Dados Portatil SQLite

## Instalação

```bash
git clone https://github.com/spadaritucks/top5-tiao-carreiro
```
## Front-end (React)
```bash
npm install
```
```bash
npm run dev
```
## Back-end (Laravel)

```bash
composer install
```
```bash
php artisan migrate --seed
```
```bash
php artisan serve
```
